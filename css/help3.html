FAQ

Thank you for visiting NewsToday! We have listed some FAQ questions below.

Q: What does NewsToday do?

A: NewsToday is an application that allows users to view top headlines from the entertainment, technology, and sports categories. NewsToday also includes a search feature that allows users to search keywords and select news categories which will then display matching results pertaining to the keyword. These results are further sorted by the user's selected category.

Q: How does the search feature work on NewsToday?

A: The search box is located at the top of the page. The user can type any keyword in the box and select categories from the drop down located to the left. After selecting categories and typing a search keyword, the user should select the search button. All matching results will then be categorized and displayed.

Q: What if I don't select a category when I search?

A: If a category is not selected upon search, your keyword will be searched without a specified category. This means that the results will not be sorted by category.

Q: What does no results found mean?

A: If you see the message, "no results found", it means that there are no results currently matching your keyword and selected categories.
